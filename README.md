# -API-
Автоматизация получения данных о погоде с API с retry, логированием и уведомлениями
🎯 Задача

Создать автоматизацию, которая:

получает данные из API (погода)
отправляет результат в Telegram
обрабатывает ошибки API
повторяет запрос при сбое (retry)
логирует ошибки в Google Sheets
⚙️ Как работает система
HTTP (запрос к API)
 ↓
если успешно → Telegram (погода)

если ошибка:
  ↓
  Sleep → повтор запроса
  ↓
  Sleep → повтор запроса
  ↓
  если ошибка → лог в Google Sheets + Telegram
🛠 Используемые инструменты
Make (Integromat)
HTTP (GET запрос)
Telegram Bot API
Google Sheets
OpenWeather API
🔥 Ключевые функции
1. Работа с API
GET запрос к OpenWeather
передача параметров (город, units, API key)
обработка JSON-ответа
2. Форматирование данных
преобразование JSON → текст
вывод температуры и описания
форматирование даты
3. Умная логика (conditions)
if / contains / lower
разные сообщения в зависимости от погоды:
дождь → ☔
облака → ☁️
ясно → ☀️
4. Retry (повтор запроса)
3 попытки при ошибке
задержка через Sleep
устойчивость к временным сбоям API
5. Обработка ошибок
error handler для HTTP
разные сценарии при ошибке
сценарий не падает
6. Логирование
запись ошибок в Google Sheets
фиксируется:
время
шаг
ошибка
описание
7. Уведомления
отправка результата в Telegram
отдельное сообщение при ошибке
📊 Результат
автоматизация работает без падений
ошибки фиксируются
пользователь получает уведомления
система устойчива к сбоям API
